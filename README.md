# gym-management

# Sistema de Academia em Java

Projeto orientado a objetos desenvolvido em Java puro
para praticar modelagem de domínio antes de usar Spring Boot.

## Conceitos aplicados

- Herança
- Abstração
- Polimorfismo
- Encapsulamento
- Composição de objetos

## Domínio

Pessoa
 ├ Aluno
 ├ AlunoVip
 └ Instrutor

Plano
 ├ PlanoStandart
 ├ PlanoVip
 └ PlanoBlack

## Funcionalidades

- cadastro de alunos
- cadastro de instrutores
- gerenciamento de planos
- agendamento de aulas
